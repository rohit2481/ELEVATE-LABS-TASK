***************************** command in use ******************************
sudo su  ---> To become super user
ufw ----> to run or if not install so the system will automatically install package but with permission  
ufw status ---> check our uncompiled firewall status
ufw enable  -----> unable if service is not active
ufw status -----> for check the status update or not
ufw status numbered  ---->  displays your active UFW rules in a sequential, numbered list
sudo ufw deny 23/tcp ----> to block telnet port
ufw status numbered ----> verify the rule 
ufw allow 22/tcp -----> allow ssh port
ufw status numbered -----> verify the rule 
ufw delete deny 23/tcp -----> remove the test rule
ufw status numbered ------> verify final check
